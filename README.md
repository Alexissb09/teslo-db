# TesloDB API

1. Clonar proyecto

2. `npm install`

3. Clonar `.env.template` y renombrarlo a `.env`

4. Cambiar variables de entorno

5. Levantar la base de datos

```
docker-compose up -d
```

6. Arrancar app: `npm start:dev`

7. Rellenar la base de datos con la seed

```
localhost:3000/api/seed
```
